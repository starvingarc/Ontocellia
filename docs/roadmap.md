# Roadmap

Ontocellia is being built as a framework-first developmental agent tissue system. The current foundation includes genome, cell, induction, LLM effector, developmental field, organ selection, communication, and validation-runner layers.

## Completed

- Phase 1: Genome layer
- Phase 2: Cell layer
- Phase 3A: Induction compiler
- Phase 3B: LLM effector layer
- Phase 4: Morphogen field, tissue topology, and fate landscape
- Phase 5: Organ selection layer
- Phase 6: Communication, handoff, and shared matrix memory
- Phase 7: Validation hook runner

## Next Phases

### Phase 8: MCP / Extracellular Interface Adapter

Map MCP servers, tools, resources, prompts, and responses onto extracellular niches, membrane channels, extracellular matrix, induction factors, receptor schemas, and returned metabolite signals.

### Phase 9: Mutation-Selection-Solidification

Turn failed attempts and lineage evidence into mutation candidates. Retain mutations only when organ selection improves validated tissue function.

### Phase 10: Complete Task Tissue Demo

Build a complete repo repair or research synthesis demonstration with induction, development, communication, LLM intents, validation results, matrix memory, and report artifacts.

## Product Direction

The reference simulation runtime can remain a developmental sandbox. The main product direction is an agent tissue orchestration framework with decentralized coordination and traceable self-repair.
